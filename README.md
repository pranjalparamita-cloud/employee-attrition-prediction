# 🎯 Employee Attrition Risk Prediction System

A professional, ML-powered analytics platform for predicting and managing employee attrition risk at Palo Alto Networks.

## 🚀 Features

- **Predictive Analytics**: ML models predict attrition risk with high accuracy
- **Risk Scoring**: Individual employee risk scores (0-100%)
- **Interactive Dashboard**: Beautiful, intuitive Streamlit interface
- **Department Analytics**: Drill-down analysis by department and role
- **Employee Profiles**: Detailed risk assessment for each employee
- **Actionable Insights**: Data-driven retention recommendations
- **Model Performance**: Transparent metrics and feature importance
